1. Make sure input folder is empty
2. Place csv file from square that you need in the input folder. Rename the file "input" NO CAPS!
3. Double click the file called batchfile, in the Square CSV folder. Command prompt will open, when its finished, it will say "pause"
4. Go to output folder to collect your new list, it will still be a CSV
5. Recommended to export the new file into google sheets to print.
